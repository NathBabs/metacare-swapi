Make Anonymous Comment

Make an anonymous comment on a star wars movie, by including thee movie id at the end of the URL

  • Curl
  • HTTP
curl -X POST -d '{
    "comment": "Comment from post man"
}' "localhost:6000/film/comment/1"
POST 6000/film/comment/1 HTTP/1.1
Host: 

{
    "comment": "Comment from post man"
}
  • Response
Status200 OK
Content-Typeapplication/json; charset=utf-8
{
    "success": true,
    "data": {
        "id": 12,
        "comment": "Comment from post man",
        "ip": "::1",
        "movie": 1,
        "createdAt": "2021-10-04T09:44:59.298Z"
    }
}

Get character names in a movie
